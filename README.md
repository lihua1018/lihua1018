### 👋 關於我 (About Me)

我擁有 **智慧機器人** 與 **小學教育** 的獨特雙學位背景。我的核心競爭力在於 **「翻譯」** —— 我擅長建構複雜的系統，更擅長設計讓人類能直觀理解與操作的介面。

- 🔭 **目前專注**：視覺語言模型 (VLM) 與機械手臂 OPC UA 串接。
- 🛠️ **硬科技 (Hard Tech)**：ROS、電腦視覺 (OpenCV)、製造系統整合。
- 🧩 **軟實力 (Soft Skills)**：運用教育學（鷹架理論、差異化教學）優化系統 UX。

> 💡 *我相信：最好的程式碼不僅要能跑，還要能被理解；最好的系統不僅要強大，還要能與人協作。*

### 🧩 核心哲學 (The Engineer-Educator Hybrid)

我的開發流程融合了教學設計原理：

| **工程思維 (Engineering)** | **教育思維 (Pedagogy)** | **我的整合 (My Approach)** |
| :--- | :--- | :--- |
| **系統設計** (System Design) | **鷹架理論** (Scaffolding) | 設計模組化系統，讓使用者能循序漸進掌握功能。 |
| **電腦視覺** (Computer Vision) | **觀察評量** (Assessment) | 讓機器人像老師觀察學生一樣，理解人類意圖。 |
| **除錯優化** (Debugging) | **差異化教學** (Differentiation) | 針對新手與專家，提供動態調整的操作回饋。 |
